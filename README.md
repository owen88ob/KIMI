## 🎯 核心体验

1. **打开页面** → 看到 Kimi 大 Logo，充满期待
2. **输入消息** → 跳转聊天界面，点击发送
3. **等待回复** → 三个跳动的圆点，紧张又期待
4. **收到回复** → `和 kimi 聊天的人太多啦，订阅会员可进入独立的优先队列～`
5. **点击任何功能** → `订阅会员才能使用`
6. **点击订阅** → `仅会员订阅用户才能订阅会员`
7. **循环往复，功德圆满** 🔄

## 🛠️ 技术栈

- **HTML5** — 整个项目只有一个文件
- **CSS3** — CSS 变量 + 主题系统，无任何 CSS 框架
- **原生 JavaScript** — 零依赖，不用 React/Vue/jQuery
- **i18n** — 内置中英文国际化
- **SVG 图标** — 全部内联，无外部图标库
